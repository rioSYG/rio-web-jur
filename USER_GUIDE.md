# RIO Academic Library - User Guide

## 🇬🇧 How to Search for Journals

### Method 1: Using Country Selector (Recommended)
1. On the homepage, you'll see two large buttons in the Hero section (top):
   - **🇮🇩 Indonesia** - Search for Indonesian local journals
   - **🌐 Global** - Search for international journals

2. **Select "Indonesia"** to search journals from local sources:
   - **GARUDA** - Indonesian scientific journal portal from Ministry of Education
   - **DOAJ Indonesia (SINTA)** - Indexed Indonesian open access journals

3. Type your search keywords and click **"Search"** or press Enter

### Method 2: Using Filters (Advanced)
1. After performing a search, click the **"Filters & Sources"** button
2. In the **"Source"** dropdown, select:
   - `GARUDA (Indonesia)` - For GARUDA journals
   - `DOAJ Indonesia (Sinta)` - For DOAJ indexed journals
   - Or leave `All sources` to search both

3. Adjust other filters (year, category, sorting) as needed

---

## 🌐 Searching for Global/International Journals

1. Click the **"Global"** button in the hero section
2. The app will search international sources:
   - **Crossref** - Multidisciplinary scholarly literature
   - **arXiv** - STEM preprints
   - **PubMed** - Biomedical and health literature

3. Type your keywords and click **"Search"**

## 🌐 Changing Language (Language Switcher)

### Where is the Language Switcher:
- **In the header (top section)**, on the right side of the navigation menu
- Appears as a button with a flag icon and text **"Bahasa"** or **"Language"**

### How to Change Language:
1. **Click the "Bahasa/Language" button** in the header
2. A dropdown menu will appear with options:
   - **🇮🇩 Indonesia**
   - **🇬🇧 English**
3. Select your preferred language
4. The page will automatically switch to the selected language

**Note:** Your language preference is saved and will persist on future visits.

## 📋 Key Features

### 1. **Search**
- Search by title, author, topic, DOI, or keywords
- Real-time search with autocomplete suggestions
- Support for both Indonesia and Global modes

### 2. **Advanced Filters**
After searching, you can refine results with:
- **Source** - Choose journal source
- **Sort By** - Sort by:
  - Relevance (default)
  - Latest (date)
  - Most cited
- **Year Range** - Filter by publication year
- **Category** - Subject category (Biology, Physics, Medicine, etc.)
- **Results per page** - Number of results displayed

### 3. **Journal Cards**
Each search result displays:
- **Title** of the journal
- **Authors** - Author names
- **Abstract** - Brief summary
- **Journal** - Publication name
- **Publication Year**
- **Citations** - Number of citations (if available)
- Action buttons:
  - ⭐ **Bookmark** - Save for later
  - 👁️ **View details** - See full information
  - 📄 **Open source** - Access journal at original source
  - 📥 **Export** - Download as PDF or DOCX

### 4. **Bookmarks**
- Click **⭐ Save bookmark** on any journal
- Access your bookmarks in the **"Bookmarks"** page (in navigation menu)
- Features on Bookmarks page:
  - View all saved journals
  - Write research notes for each journal
  - Export bookmarks in formats:
    - JSON
    - CSV
    - BibTeX (for reference management)
  - Delete unwanted bookmarks

### 5. **Journal Detail Page**
Click **"View details"** to see:
- Complete journal information
- Full list of all authors
- Complete abstract
- Keywords and category
- Identifiers: DOI, PMID, arXiv ID
- Your research notes
- Link to original source
- Export and bookmark options

### 6. **Export**
Export journals in multiple formats:
- **PDF** - For offline reading
- **DOCX** - For editing in Word
- **BibTeX** - For reference management

## ❓ Troubleshooting

### No Results in Indonesia Mode
**Problem:** Indonesia search returns no results
**Solution:**
1. Make sure you've selected **"Indonesia"** in the country selector
2. Try different search keywords (more general/specific)
3. Expand the year range in the search
4. If still no results, try **"Global search"** (will include international journals)

### No Results in Global Mode
**Problem:** Global search returns no results
**Solution:**
1. Check the spelling of your keywords
2. Use more general or more specific keywords
3. Expand the year range (default: last 5 years)
4. Try different sources in the filter (Crossref, arXiv, PubMed)

### Language Switcher Not Visible
**Problem:** Cannot find the language switching button
**Solution:**
1. Look for a button with a **flag icon** or text **"Bahasa"/"Language"** in the top right
2. On mobile, the button might be in the hamburger menu (≡)
3. Click once (don't hover) to open language options

### Bookmarks Not Saved
**Problem:** Bookmarks disappear after closing browser
**Solution:**
1. Bookmarks are stored in your device's **Local Storage**
2. Don't clear browsing data/cookies
3. For permanent sync, login with your account using **"Demo Login"** or **"GitHub"** button

## 💡 Tips & Tricks

### Effective Searching
- 🔍 Use **specific keywords** for more accurate results
- 📅 Use **Year Range filter** to find the latest publications
- 🏷️ Use **DOI** or **PubMed ID** for precise search
- 🌍 Try **Global search** first if Indonesian journals are unavailable

### Bookmark Management
- 📝 Write **research notes** for each bookmark to remember key points
- 📊 Export to **BibTeX** if you use Zotero or Mendeley
- 📱 Bookmarks are accessible on the same device (Local Storage)

### Navigation
- 🔄 Use **"Reset search"** button to start fresh
- ⬅️ **"Back"** button takes you to the search page
- ⭐ **"Bookmarks"** menu in navbar to view all saved journals
