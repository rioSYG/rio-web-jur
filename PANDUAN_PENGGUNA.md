# Panduan Pengguna RIO Academic Library

## 🇮🇩 Mencari Jurnal Indonesia

### Cara 1: Menggunakan Country Selector (Recommended)
1. Di halaman utama, Anda akan melihat dua tombol besar di bagian Hero (atas):
   - **🇮🇩 Indonesia** - Mencari jurnal lokal Indonesia
   - **🌐 Global** - Mencari jurnal internasional

2. **Pilih "Indonesia"** untuk mencari jurnal dari sumber lokal:
   - **GARUDA** - Portal jurnal ilmiah Indonesia dari Kemdikbud
   - **DOAJ Indonesia (SINTA)** - Daftar Jurnal Open Access terindeks internasional

3. Ketik kata kunci pencarian Anda dan tekan **"Cari"** atau Enter

### Cara 2: Menggunakan Filter (Advanced)
1. Setelah melakukan pencarian, klik tombol **"Filter & Sumber"**
2. Di bagian **"Source"**, pilih:
   - `GARUDA (Indonesia)` - Untuk jurnal GARUDA
   - `DOAJ Indonesia (Sinta)` - Untuk jurnal DOAJ terindeks
   - Atau biarkan `Semua sumber` untuk kedua sumber

3. Sesuaikan filter lainnya (tahun, kategori, urutan) sesuai kebutuhan

---

## 🌐 Mencari Jurnal Global/Internasional

1. Pilih tombol **"Global"** di hero section
2. Aplikasi akan mencari di sumber internasional:
   - **Crossref** - Literatur ilmiah lintas disiplin
   - **arXiv** - Preprint di bidang STEM
   - **PubMed** - Literatur biomedis dan kesehatan

3. Ketik kata kunci dan tekan **"Cari"**

## 🌐 Mengubah Bahasa (Language Switcher)

### Lokasi Language Switcher:
- **Di header (bagian atas)**, di sebelah kanan navigation menu
- Terlihat sebagai tombol dengan ikon bendera dan teks **"Bahasa"** atau **"Language"**

### Cara Mengubah Bahasa:
1. **Klik tombol "Bahasa/Language"** di header
2. Sebuah dropdown menu akan muncul dengan pilihan:
   - **🇮🇩 Indonesia**
   - **🇬🇧 English**
3. Pilih bahasa yang Anda inginkan
4. Halaman akan otomatis berubah ke bahasa yang dipilih

**Catatan:** Pilihan bahasa Anda akan disimpan dan tetap aktif di kunjungan berikutnya.

## 📋 Fitur-Fitur Utama

### 1. **Search (Pencarian)**
- Cari berdasarkan judul, penulis, topik, DOI, atau kata kunci
- Pencarian real-time dengan saran otomatis
- Dukungan untuk Indonesia dan Global mode

### 2. **Advanced Filters (Filter Lanjut)**
Setelah pencarian, Anda dapat menyaring dengan:
- **Source/Sumber** - Pilih sumber jurnal
- **Sort By** - Urutkan berdasarkan:
  - Relevansi (default)
  - Terbaru (date)
  - Sitasi terbanyak
- **Year Range** - Filter berdasarkan tahun publikasi
- **Category** - Kategori ilmu (Biologi, Fisika, Kedokteran, dll)
- **Results per page** - Jumlah hasil per halaman

### 3. **Journal Cards (Kartu Jurnal)**
Setiap hasil pencarian menampilkan:
- **Judul** jurnal
- **Penulis** (nama-nama penulis)
- **Abstrak** (ringkasan singkat)
- **Jurnal** - Nama publikasi
- **Tahun publikasi**
- **Sitasi** - Jumlah kutipan (jika tersedia)
- Tombol aksi:
  - ⭐ **Bookmark** - Simpan untuk dibaca nanti
  - 👁️ **Lihat detail** - Lihat informasi lengkap
  - 📄 **Buka sumber** - Akses jurnal di sumber asli
  - 📥 **Export** - Download PDF atau DOCX

### 4. **Bookmarks (Penyimpanan)**
- Klik tombol **⭐ Simpan bookmark** pada setiap jurnal
- Akses bookmark Anda di halaman **"Bookmark"** (di menu navigasi)
- Fitur pada halaman Bookmark:
  - Lihat semua jurnal yang disimpan
  - Tulis catatan riset untuk setiap jurnal
  - Ekspor bookmark ke format:
    - JSON
    - CSV
    - BibTeX (untuk referensi)
  - Hapus bookmark yang tidak perlu

### 5. **Detail Journal (Halaman Detail)**
Klik **"Lihat detail"** untuk melihat:
- Informasi lengkap jurnal
- Daftar semua penulis
- Abstrak lengkap
- Keywords dan kategori
- Identifikasi: DOI, PMID, arXiv ID
- Catatan riset Anda
- Link ke sumber asli
- Opsi untuk ekspor atau bookmark

### 6. **Export (Ekspor)**
Ekspor jurnal dalam beberapa format:
- **PDF** - Untuk membaca offline
- **DOCX** - Untuk editing di Word
- **BibTeX** - Untuk manajemen referensi

## ❓ Troubleshooting

### Tidak Ada Hasil di Mode Indonesia
**Masalah:** Pencarian Indonesia tidak menampilkan hasil
**Solusi:**
1. Pastikan Anda sudah memilih **"Indonesia"** di country selector
2. Coba ganti kata kunci pencarian (lebih umum/spesifik)
3. Perluas rentang tahun pencarian
4. Jika tetap tidak ada hasil, coba **"Global search"** (akan mencakup jurnal internasional)

### Tidak Ada Hasil di Mode Global
**Masalah:** Pencarian global tidak menampilkan hasil
**Solusi:**
1. Cek ejaan kata kunci
2. Gunakan kata kunci yang lebih umum atau lebih spesifik
3. Perluas rentang tahun (default: 5 tahun terakhir)
4. Coba sumber lain di filter (Crossref, arXiv, PubMed)

### Language Switcher Tidak Terlihat
**Masalah:** Tidak bisa menemukan tombol untuk mengubah bahasa
**Solusi:**
1. Cari tombol dengan **ikon bendera** atau teks **"Bahasa"/"Language"** di bagian atas kanan halaman
2. Jika di mobile, tombol mungkin di menu hamburger (≡)
3. Klik sekali (bukan hover) untuk membuka pilihan bahasa

### Bookmark Tidak Tersimpan
**Masalah:** Bookmark hilang setelah menutup browser
**Solusi:**
1. Bookmark disimpan di **Local Storage** perangkat Anda
2. Jangan hapus data browsing/cookies
3. Untuk sinkronisasi permanent, login dengan akun Anda di tombol **"Demo Login"** atau **"GitHub"**

## 💡 Tips & Trik

### Pencarian Efektif
- 🔍 Gunakan **kata kunci spesifik** untuk hasil lebih akurat
- 📅 Manfaatkan filter **Year Range** untuk menemukan publikasi terbaru
- 🏷️ Gunakan **DOI** atau **PubMed ID** untuk pencarian presisi
- 🌍 Coba **Global search** terlebih dahulu jika Indonesia tidak ada

### Manajemen Bookmark
- 📝 Tulis **catatan riset** di setiap bookmark untuk mengingatkan key points
- 📊 Ekspor ke **BibTeX** jika Anda menggunakan Zotero atau Mendeley
- 📱 Bookmark dapat diakses di device yang sama (Local Storage)

### Navigasi
- 🔄 Gunakan tombol **"Reset pencarian"** untuk mulai dari awal
- ⬅️ Tombol **"Kembali"** membawa Anda ke halaman pencarian
- ⭐ Menu **"Bookmark"** di navbar untuk melihat semua penyimpanan Anda
